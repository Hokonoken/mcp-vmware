# Support

This is a personal, best-effort open-source project. There is **no commercial
support and no SLA** (see the Disclaimer in the [README](README.md)).

That said, here is how to get help:

- **Questions / usage**: open a [GitHub Discussion][discussions] or an issue
  with the `question` label.
- **Bugs**: open an [issue][issues] using the bug template.
- **Feature ideas**: open an [issue][issues] using the feature template.
- **Security vulnerabilities**: do **not** open a public issue — see
  [SECURITY.md](SECURITY.md).

Before opening an issue, please check `api-map/coverage.yaml` (the wanted tool
may already be planned) and the existing open issues.

Response is on a best-effort basis, on the maintainer's own time.
